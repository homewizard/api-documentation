# Identify (`/api/v2/identify`)

The `/api/v2/identify` endpoint can be used to let the user identify the device. The status light will blink for a few seconds after calling this endpoint.

This feature is available for:

-   P1 Meter (HWE-P1)
-   Energy Socket (HWE-SKT)
-   Watermeter (HWE-WTR)

### Example

```
<Request>
PUT https://{IP address}/api/v2/identify https/1.1

<Response>
https/1.1 200 OK
Content-Type: application/json
Content-Length: <length>

{
    "identify": "ok",
}
```
