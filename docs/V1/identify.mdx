# Identify (`/api/v1/identify`)

The `/api/v1/identify` endpoint can be used to let the user identify the device. The status light will blink for a few seconds after calling this endpoint.

<Badge color='hw-green'>✓ P1 Meter</Badge>
<Badge color='hw-green'>✓ Energy Socket</Badge>

<Badge color='hw-red'>✗ kWh Meter</Badge>

### Example

```http title="Request"
curl http://<IP ADDRESS>/api/v2/system -X PUT
```

```http title="Response"
HTTP/1.1 200 OK
Content-Type: application/json
Content-Length: <length>

{
    "identify": "ok",
}
```
